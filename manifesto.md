# Manifesto

> Events first. Provenance always. JSON as truth.

## Thesis

News systems for AI agents should not start from pages. They should start from facts under evidence.

Humans want a cleaner workspace.
Agents want a cleaner contract.
The system should provide both without splitting the truth boundary.

## Principles

1. **Events over articles.** Articles are inputs. Events are the primary unit of meaning.
2. **JSON before Markdown.** Typed records are the source of truth. Markdown is a projection.
3. **Provenance is not optional.** Every meaningful claim must name its supporting sources.
4. **Rights are part of the model.** Storage and output rules depend on source permissions, not convenience.
5. **Human UI rides on the same core.** The interface for people must read the same event layer exposed to agents.
6. **Narrow scope is strength.** Crypto first. One trustworthy path before platform sprawl.
7. **Signals beat noise.** Deduplication, clustering, and confidence matter more than feed volume.
8. **Summaries are derived artifacts.** Generated summaries are useful, but they do not replace evidence.

## Anti-Principles

1. **No Markdown-only truth model.**
2. **No scrape-first product posture.**
3. **No source ingestion without rights review.**
4. **No hand-wavy event claims without stable IDs and timestamps.**
5. **No generic media product drift before agent contracts are sound.**

## Claim Boundary

Sift should claim only this:

- source records were collected and normalized;
- related records were clustered into events;
- provenance and rights metadata were preserved;
- summaries and digests were derived from typed records;
- human and agent surfaces reflected the same underlying event layer.

It should not claim:

- universal factual completeness;
- perfect clustering;
- market correctness;
- legal permission beyond the recorded source policy.
